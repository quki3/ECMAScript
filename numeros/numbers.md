# numbers
# Math.abs()
```bash
La función Math.abs(x) retorna el valor absoluto/modulo de un número
Parametros:
x //? Un número.
----------------------------------------------------------------
Math.abs('-1');     // 1
Math.abs(-2);       // 2
Math.abs(null);     // 0
Math.abs('');       // 0
Math.abs([]);       // 0
Math.abs([2]);      // 2
Math.abs([1,2]);    // NaN
Math.abs({});       // NaN
Math.abs('string'); // NaN
Math.abs();         // NaN
```
